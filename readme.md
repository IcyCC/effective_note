# Effective Node - 2019 本科生秋招后端进阶笔记

> 非入门, 进阶之路

作为一个大三的学生, 马上就面临秋招找工作的问题, 这时候发现自己,对很多知识理解太过表面, 都停留在会用的层次. 而更进阶的的用法和原理却不太知晓. 所以打算用一段时间完成会用到进阶的过程, 把其中的学习内容记录成笔记.目标是通过一段时间学习具备对后端基础设施有进阶程度的了解并具备开发中间件的能力.

本次复习笔记记录内容主要包括:

- mysql
- 缓存
- golang
- 数据结构算法
- 容器
- 计算机科学通识

因为时间舍弃但很想添加的内容(顺序有优先级):

- 网络协议
- 消息队列
- 操作系统
